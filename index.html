Today lets look at two things that are almost the same in CSS. (but not really)

Both define a group of styles that can be selected to be applied together to an element.

A CSS Class

  • Used where the style group must be applied to multiple elements
  • Defined with .
  • Called with class="xxxx"

A CSS ID

  • Used to target one particular element with the style group
  • Defined with #
  • id="xxxx"
